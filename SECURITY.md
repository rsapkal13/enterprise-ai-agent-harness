# Security Policy

Enterprise AI Agent Harness is currently a specification-stage project. It should not be used as a production control plane yet.

## Reporting Issues

Please report security concerns through the repository security advisory process when available, or by opening a private maintainer communication channel. Do not publish exploitable details in a public issue.

## Sensitive Information

Do not commit credentials, customer records, private system names, internal network details, proprietary policies, or non-public architecture diagrams. Examples must use fictional data only.
