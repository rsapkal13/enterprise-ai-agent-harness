# Contributing

Thanks for helping shape Enterprise AI Agent Harness. The project is currently focused on a clean public specification before runtime implementation.

## Contribution Priorities

- Improve the vocabulary and model boundaries.
- Strengthen JSON schemas and examples.
- Add organisation-neutral enterprise journey examples.
- Clarify governance, audit, privacy, and evaluation guidance.
- Keep runtime code minimal until the specification stabilizes.

## Ground Rules

- Do not add credentials, customer data, internal architecture, or private strategy.
- Keep examples fictional and reusable.
- Prefer vendor-neutral language unless describing an optional adapter pattern.
- Use clear Markdown and small pull requests.

## Development Notes

The repository intentionally avoids unnecessary dependencies for the v0.1 spec release. TypeScript files are placeholders for future packages and should not grow into a runtime until the related schema and design docs are stable.
