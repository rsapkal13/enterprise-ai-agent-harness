# Phase 2: Local Runtime

Goal: add a small local runtime preview.

## Scope

- Load registry files
- Validate schemas
- Register agents, skills, tools, policies, workflows, systems, context scopes, audit events, evaluations, and UI manifests
- Simulate the telco journey using mock tools

## Exit Criteria

- Demo runner can execute a dry-run journey locally
- No production integration requirements
- Clear extension points for adapters
