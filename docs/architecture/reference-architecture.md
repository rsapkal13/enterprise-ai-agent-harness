# Reference Architecture

```text
Channels
  Web, mobile, contact center, chat, voice, internal tools

AI Runtime
  Agents, assistants, planners, orchestrators

Enterprise AI Agent Harness
  Skill Registry
  Tool Registry
  Agent Registry
  Policy Engine
  Workflow Engine
  System Registry
  Context Layer
  UI Manifest Registry
  Audit, Observability, and Evaluation

Enterprise Systems
  CRM, billing, identity, case management, data platforms, knowledge stores
```

The reference architecture is intentionally generic. Implementations can map these layers onto existing platforms, managed services, or local components.
